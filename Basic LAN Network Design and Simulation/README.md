# Basic LAN Network Design and Simulation

## Project Overview

This project demonstrates the design and implementation of a basic Local Area Network (LAN) using Cisco Packet Tracer.

## Tools Used

* Cisco Packet Tracer
* Cisco 2960 Switch
* PCs

## Project Tasks

* Designed a small LAN topology using a Cisco 2960 switch and multiple PCs.
* Configured static IPv4 addresses on end devices.
* Verified network connectivity using ICMP ping tests.
* Tested both correct and incorrect IP configurations to understand connectivity behavior.
* Performed basic network troubleshooting.

## Skills Demonstrated

* LAN Configuration
* IPv4 Addressing
* Network Connectivity Testing
* ICMP Ping
* Basic Network Troubleshooting
